Generate sequence from VDM

VDMToolsで実行したログからシーケンス図を生成するastahプラグイン

Download this project as a .zip file Download this project as a tar.gz file

プラグイン概要

VDMTools v9.0.3以降に追加された、実行ログ取得機能で出力されたログをインプットに、 astah* Professionalを用いて実行シーケンス図を作成するプラグインです。

インストール方法

  • astah* Professionalを開き、 ヘルプ -> プラグイン一覧 を選択
  • プラグイン一覧画面の左上部にある インストール ボタンを押下
  • 以下のダウンロードからダウンロードした generate_sequence_fromVDM-1.0.0.jar を選択
  • astah* Professionalを再起動

使い方

  • VDMToolsのインタープリタにて右のコマンドを発行 >> set calllog
  • インタープリタにて実行
  • 実行終了後、作成されたcalllog.txtからシーケンス図を生成したい箇所をトリミング
  • astah* Professionalにて、 ツール -> VDM連携 -> ログからシーケンス図生成 を選択

ダウンロード

generate_sequence_fromVDM-1.0.0.jar
